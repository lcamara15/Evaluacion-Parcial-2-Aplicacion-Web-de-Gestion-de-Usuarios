# Sistema de Autenticación y Gestión de Perfiles:

Este proyecto consiste en una aplicación web full stack que permite la autenticación de usuarios y la gestión de información personal mediante un módulo de perfil.

<img width="1365" height="678" alt="image" src="https://github.com/user-attachments/assets/b3d47d34-ed0f-494e-bd5b-03ebb4efc288" />

La solución fue desarrollada utilizando Angular en el frontend, Node.js con Express en el backend y PostgreSQL como base de datos, integrando una arquitectura cliente-servidor mediante una API REST.

## Funcionalidades principales:

- Inicio de sesión mediante usuario o correo y contraseña
- Validación de credenciales contra la base de datos
- Creación de perfil de usuario
- Consulta de información personal
- Actualización de datos del perfil

## Estructura del proyecto:

El repositorio contiene dos partes principales:

- **backend/**: encargado de la lógica del servidor, manejo de rutas y conexión con la base de datos
- **frontend/**: interfaz de usuario desarrollada en Angular

## Ejecución del proyecto:

### Backend:
Instalar dependencias:

npm install

Iniciar el servidor:

npm run dev

### Frontend:
Instalar dependencias:

npm install

Ejecutar aplicación:

ng serve

## Tecnologías utilizadas:

- Angular
- Node.js
- Express
- PostgreSQL
